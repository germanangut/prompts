# Planning Agents

Agent definitions and system prompts for planning and coordination AI agents.

## When to Use These Agents

Deploy a planning agent when you need to:
- Break down complex goals into actionable tasks
- Create project plans and timelines
- Prioritize a backlog of work
- Identify dependencies and critical paths
- Track and update plans as circumstances change

## Typical Capabilities

- Task decomposition and dependency mapping
- Timeline and resource estimation
- Priority scoring (impact vs. effort)
- Risk identification and mitigation planning
- Progress tracking and plan updates
