# Coding Agents

Agent definitions and system prompts for software engineering AI agents.

## When to Use These Agents

Deploy a coding agent when you need to:
- Generate, review, or refactor code autonomously
- Write tests for a codebase
- Debug issues given error messages and code
- Implement features from specifications
- Perform code migrations or upgrades

- Test writing and execution
- Debugging and root cause analysis
- Documentation generation
