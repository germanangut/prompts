# Research Agents

Agent definitions and system prompts for research-focused AI agents.

## When to Use These Agents

Deploy a research agent when you need to:
- Search and synthesize information from multiple sources
- Fact-check claims or find supporting evidence
- Generate literature reviews or competitive analyses
- Monitor topics for new developments
- Answer questions that require up-to-date information
