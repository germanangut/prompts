# Orchestrator Agents

Agent definitions and system prompts for orchestrator agents that coordinate other agents.

## When to Use These Agents

Deploy an orchestrator agent when you need to:
- Coordinate a team of specialist agents on a complex task
- Route incoming tasks to the right specialist
- Synthesize outputs from multiple agents into a coherent result
- Manage workflows that span multiple agent capabilities
- Handle error recovery and retries across a multi-agent system

## Typical Capabilities

- Task intake, parsing, and classification
- Agent selection and delegation
- Parallel execution coordination
- Result synthesis and quality checking
- Escalation to human when needed
