# Coding Prompts

- **Code generation** — generating boilerplate, functions, classes, APIs
- **Code review** — reviewing code for quality, bugs, and best practices
- **Debugging** — diagnosing and fixing bugs
- **Refactoring** — improving existing code structure and readability
- **Testing** — writing unit, integration, and end-to-end tests
