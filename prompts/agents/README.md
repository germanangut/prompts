# Agent Prompts

System prompts, configuration templates, and meta-prompts for AI agents and multi-agent systems. Use these when you need to:

- Configure a specialist AI agent with a role and constraints
- Define agent capabilities and boundaries
- Set up multi-agent communication protocols
- Create agent evaluation and self-reflection prompts

## Files in This Directory

| File | Description |
|---|---|
| [`system-prompt-template.md`](./system-prompt-template.md) | Base template for any agent system prompt |
| [`agent-role-definition.md`](./agent-role-definition.md) | Define an agent's role, goals, and constraints |
| [`multi-agent-coordinator.md`](./multi-agent-coordinator.md) | Coordinate tasks across multiple specialized agents |
| [`agent-self-reflection.md`](./agent-self-reflection.md) | Prompt an agent to evaluate and improve its own output |
