# General Prompts

Cross-domain, general-purpose prompts that apply across many contexts. Use these when:

- You need to brainstorm ideas or solutions
- You are planning a project or task
- You need to make a decision with multiple options
- You want to ask clarifying questions before diving in
- You need a structured QA or review pass
