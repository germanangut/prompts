# Automation Projects

Prompts tailored for automation, RPA, and workflow engineering projects.

## When to Use These Prompts

Use this directory when you are working on:
- Business process automation
- Robotic process automation (RPA)
- Integration pipelines between SaaS tools
- Scheduled jobs and cron-based workflows
- Chatbot and conversational automation
